# data engineering mini pipeline
This project reads book data from , cleans it, stores it in a SQLite database, and visualizes the top 5 most expensive books using Python, Pandas, and Seaborn.

## Features
-Web scraping data from books.toscrape.com
-Cleaning and preprocessing of data
-Storing the data in a SQLite database
-Simple analysis to extract insights (top 5 expensive books)

## Technologies
- Python 3
- Pandas
- SQLite
- Matplotlib
- Seaborn

## Project Outputs:
raw_books.csv → raw scraped data
cleaned_books.csv → cleaned dataset
database.db → SQLite database
top_books.csv → analysis results
